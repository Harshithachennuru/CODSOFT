# 🎮 Tic-Tac-Toe AI 🤖

This project is developed as part of my **Artificial Intelligence Internship at CODSOFT**.

It is a console-based Tic-Tac-Toe game where a human player competes against an AI that uses the **Minimax Algorithm**, making it unbeatable.

---

## 📌 Features

- 👤 Human vs AI gameplay  
- 🧠 AI uses Minimax Algorithm  
- 🚫 Unbeatable AI (always wins or draws)  
- 💻 Simple console-based interface  

---

## 🛠️ Technologies Used

- Java  
- Core concepts of Artificial Intelligence  
- Game Theory (Minimax Algorithm)  

---

## 🧠 How It Works

The AI uses the **Minimax Algorithm** to evaluate all possible moves:

- ✔️ Maximizes its chances of winning  
- ❌ Minimizes the opponent’s chances  
- 🎯 Chooses the optimal move every time
