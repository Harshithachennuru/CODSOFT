# Image Captioning AI (CODSOFT Task 3)

## 📌 Project Overview

This project demonstrates a simple Image Captioning system using a pre-trained deep learning model.
It takes an image as input and generates captions based on detected objects.

---

## 🛠️ Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Pillow

---

## 📂 Project Structure

CODSOFT/
├── caption.py
├── image.jpg
└── README.md

---

## ▶️ How to Run

1. Install dependencies:
   pip install tensorflow numpy pillow

2. Run the program:
   python caption.py

---

## 📷 Input

Provide an image named **TASK#_ImageCaptioning.jpg** in the same folder.

---

## 📌 Output Example

dog (0.85)
animal (0.10)
pet (0.05)

---
